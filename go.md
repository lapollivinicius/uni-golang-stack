# Go Lang 

this is my note about go lang

to start a project in go, you need install go language

after install, we can create our first go file and run it using:

```
go run <file>
```

### modules 

In go we have module, and we can start a module using :

```
go mod init <module_name>
```

this is similar to python module, all files must be in this dir/path
